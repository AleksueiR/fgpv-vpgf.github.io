app.common.router Module

The common.router provides support to handling ui.router states and transitions.

provider

  • statehelperConfig - The `statehelperConfig` provider is used to configure `statehelper` before it's initialized. Must configure via the statehelperConfigProvider in core.config.js

service

  • statehelper - The `statehelper` services supports adding additional states to the global state tree from other modules. It is also responsible for handling routing errors (coming soon).
