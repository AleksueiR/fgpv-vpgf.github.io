app.ui.panels Module

The app.ui.panels module contains all the panel and pane templates and directives.

directive

  • rvContentPane - The `rvContentPane` directive is a panel inner container holding the panel's content.
  • rvFiltersPanel - The `rvFiltersPanel` directive is a outer panel container with a content plug view to allow for different content to be displayed.
  • rvMainPanel - The `rvMainPanel` directive is a outter panel container with a content plug view to allow for different content to be displayed.
  • rvSidePanel - The `rvSidePanel` directive is an side panel outter container with a content plug view to allow for different content to be displayed.
