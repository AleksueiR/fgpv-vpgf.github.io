app.ui.toc Module

The app.ui.toc module contains toc (or layer selector) panel.

directive

  • rvLayerGroupToggle - The `rvLayerGroupToggle` directive is a toggle for a group of layer items.
  • rvLayerItem - The `rvLayerItem` directive is a UI compoenent for a layer in the layer selector (toc).

function

  • TocController - The `TocController` controller handles the layer selector (or toc) main panel. Right now it's hacked together for demo purposes. `TocController` has lots of ugly code to handle state switching. Should be rewritten.

service

  • tocService - The `tocService` service provides bindable layer data to the `TocController`'s template. __Some hardcoded sample config data.__
