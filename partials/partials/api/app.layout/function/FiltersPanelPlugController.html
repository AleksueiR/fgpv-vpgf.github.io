FiltersPanelPlugController

 (function)

The FiltersPanelPlugController controller handles the filters panel plug view. self.active is triggering an active CSS class to be added to the side panel plug when it's active. self.mode is bound to a CSS class indicating current mode of the filters panel (default, attached, minimized, or full). rvMorph directive is used to animate between modes (when rv-morph attribute is applied to a HTML node).
